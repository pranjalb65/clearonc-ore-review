Every clinical decision and every flag is deterministic and cited (green). A language model is used only to read the record, ground wording in the literature, and write the explanation (amber). It never makes or overrides a decision. All steps are shown. Only the amber ones use a language model, and none of them is a decision.

This tree is the ORE architecture; the same steps apply to any cancer type. Example: 62F, invasive ductal carcinoma, cT2N1M0 Stage IIB, ER+/PR+, HER2 IHC 3+, ECOG 1, on trastuzumab plus paclitaxel, DPYD untested, no baseline LVEF.
